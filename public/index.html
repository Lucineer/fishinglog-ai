How It Works

From camera to logbook in seconds

1

See

Two IP67 cameras capture the deck and catch area. Frames selected at 2Hz, classified locally at 15 FPS by YOLOv8-nano.

Camera → Frame Selection → YOLOv8-nano (FP16) → Detections
2

Hear

Waterproof headset with noise suppression. Wake word "Hey Cap" activates command recognition powered by local Whisper-tiny.

Mic → RNNoise → Wake Word → Whisper-tiny → Intent → Action
3

Learn

Captain corrections create training triplets. Uploaded to cloud when connected. Next OTA update improves the edge model.

Correction → (image, wrong_label, correct_label) → Cloud Retrain → OTA Update
4

Report

Catch records auto-generated from vision + voice. Formatted for regulatory submission. Stored locally, synced when online.

Detection + Confirmation → Catch Record → SQLite → Auto-sync → NOAA/DFO Format
